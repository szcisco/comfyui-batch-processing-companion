温馨提示
面向非编程用户：
请勿修改本文件夹内的 .json 和 .ini 配置文件，错误修改会导致程序无法正常运行。

面向编程开发者：
您可对上述文件进行自定义修改，但需严格遵循以下规则：
1. 当修改 API 文件的加载配置（load）或新增模型后，需重新导出 API 文件并放置于当前目录，再基于更新后的 API 文件生成对应的 .ini 配置文件，确保程序能正确识别修改内容。
2. 若修改了关键参数，必须同步更新对应的插件编号（如 node #97），否则程序会因无法识别插件 ID 而报错。
